I've learned about...

Topological Data Analysis in Genomics : I developed a user-friendly R package that integrates genomic and transcriptomic data to identify cancer-associated genes. It is based on the Mapper algorithm. Read more about it in this preprint. [Dr. Pablo Camara]

B cell tolerance checkpoint integrity : The origin of problematic B cells in autoimmune disease is a longstanding question. Several autoimmune diseases – rheumatoid arthritis, type I diabetes, and systemic lupus erythematosus – have been shown to have global breaks in central and peripheral tolerance checkpoints, which leads to an accumulation of autoreactive B cells. I helped a graduate student explore if this pattern of autoimmunity held true in pemphigus vulgaris. Manuscript in prep. [Dr. Aimee Payne]
